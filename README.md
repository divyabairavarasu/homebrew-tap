# Safe Agent Homebrew Tap

This repository hosts the Homebrew formula for Safe Agent.

## Install

```bash
brew tap safeagent/tap
brew install safeagent
safeagent
```

## Update Formula

1. Publish a release in the main repo.
2. Update `Formula/safeagent.rb` with:
   - `version`
   - `sha256` for arm64/amd64
3. Commit and push.
